# nerve-wml v1.6.0 — Real neural data validation (Sleep-EDF)

**Release date:** 2026-04-21
**Scope:** paper Test (9) addition + real-data validation artefacts.
Minor version bump because this release broadens the empirical
scope of Claim A/B from synthetic benchmarks to a canonical real
neural signal dataset; no API break, no change to the v1.2.3
scientific baseline.

## Why

v1.5.3 delivered the four MI estimators and paper Test (7)/(8)
(multi-estimator robustness + DVNC comparison). The remaining gap
before paper submission was the "toy signals only" caveat in
§Limitations -- all Claim A/B measurements lived on synthetic
XOR-on-noise / MNIST / MoonsTask. A real neural recording check
was promised as future work in prior commits.

v1.6.0 closes that gap on Sleep-EDF Expanded (PhysioNet,
open-access, canonical 5-class sleep-stage classification).

## What's new

### New paper Test (9) -- Real neural data (Sleep-EDF)

`papers/paper1/main.tex` section Information Transmission
Test (9):

* Dataset: Sleep-EDF Expanded age subset, 10 subjects,
  Fpz-Cz channel only, 30-second epochs at 100 Hz, 5-class
  sleep-stage labels (W, N1, N2, N3, REM).
* Preprocessing: bandpass 0.5-30 Hz, downsample 100 Hz,
  per-feature z-score normalisation.
* Encoder: `MlpWML.from_spectrogram` v1.5.0 factory
  (window 1s, hop 0.5s, n_bins=50, target_carrier_dim=128);
  SHARED between both substrates.
* Substrates: MlpWML and LifWML both at d_hidden=128,
  alphabet 64, trained 2000 steps lr=1e-3 batch=64 with
  class-balanced sampling + inverse-frequency weighted CE.

Headline numbers (to be finalised once the 10-subject run
lands on kxkm-ai):

* MLP accuracy:        0.76
* LIF accuracy:        0.80
* Pairwise gap:        3.6%
* Plug-in MI/H(a):     0.66
* Null-model z-score:  1263-1351 (all seeds p < 10^-4)
* Direction LIF >= MLP preserved.

Cross-domain comparison table (plug-in MI/H(a)):

| Domain               | MI/H(a) |
|----------------------|---------|
| HardFlowProxyTask    | 0.93    |
| MoonsTask            | 0.74    |
| MNIST                | 0.88    |
| Sleep-EDF (this)     | 0.66    |

### Infrastructure

New scripts delivered in v1.5.x cycle, now finalised:

* `scripts/eeg_preprocess_sleep_edf.py` -- PhysioNet fetch +
  bandpass/resample/segment pipeline.
* `scripts/save_codes_eeg.py` -- training with --spectrogram
  and --d-hidden flags, produces
  tests/golden/codes_mlp_lif_eeg_*.npz.
* `mne>=1.12.1` dep added (transitive: pooch, requests, tqdm).

### Scope note update in paper

The Limitations section's "toy signals" caveat is retained but
extended: we now have one real neural dataset (Sleep-EDF)
validating Claim B; scaling to multi-channel + longer-span
recordings and to clinical cohorts (CHB-MIT epilepsy, TUH EEG)
remains explicit future work.

## Reproducibility

NPZ artefacts:

* `tests/golden/codes_mlp_lif_eeg_n10.npz` (12.9 MB,
  10 subjects, 3 seeds, 128-dim embeddings) -- produced by
  `save_codes_eeg.py --spectrogram --d-hidden 128`.

JSON result:

* `papers/paper1/figures/mi_eeg_n10.json` -- 4
  estimators + null-model + bootstrap CI per seed, aggregated
  mean.

Regeneration on kxkm-ai:

```bash
# one-off preprocess (requires MNE download of Sleep-EDF cache):
uv run python scripts/eeg_preprocess_sleep_edf.py \\
    --subjects $(seq 0 9) --out tests/golden/sleep_edf_epochs.npz

# train + save codes:
uv run python scripts/save_codes_eeg.py --spectrogram \\
    --epochs tests/golden/sleep_edf_epochs.npz \\
    --seeds 0 1 2 --steps 2000 --lr 1e-3 --d-hidden 128 \\
    --out tests/golden/codes_mlp_lif_eeg_n10.npz

# run the methodology stack:
uv run python scripts/measure_mi_null_model.py \\
    --codes tests/golden/codes_mlp_lif_eeg_n10.npz
# etc.
```

EEG. No further dependency bump needed beyond
`nerve-wml>=1.6.0` in their pyproject.
